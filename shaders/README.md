# Shaders

OpenGL + GLSL で実装したシェーダーデモ集。
WSL (Linux) 上で MinGW クロスコンパイルし、Windows で実行する構成。

## ビルド方法 (共通)

```bash
cd shaders/<project>
mkdir gen && cd gen
cmake ..
make
```

ビルド成果物は `gen/` 以下に生成される。`.exe` を Windows 側で実行。
GLSL ファイルを変更した場合も `make` だけで再コピーされる。

## プロジェクト一覧

### cube

回転する立方体を描画するシンプルなシェーダー。

- **レンダリング方式**: ラスタライズ (通常の頂点/フラグメントパイプライン)
- **ライティング**: Phong シェーディング (Ambient / Diffuse / Specular)
- **ジオメトリ**: 法線付き 36 頂点 (6面 x 2三角形 x 3頂点)
- **操作**: ESC で終了

### glass_sphere

ガラス玉・金属ブロック・草ブロック・スポットライトを含むシーンをリアルタイムレイトレーシングで描画。

- **レンダリング方式**: フラグメントシェーダー内ソフトウェアレイトレーシング (フルスクリーンクワッド)
- **マテリアル**:
  - **ガラス球** - 屈折 (IOR 1.52) / 反射 / フレネル / 全内部反射 / ガラス内吸収
  - **金属ブロック (Gold)** - 磨き仕上げの金色鏡面反射
  - **金属ブロック (Steel)** - クロム調の鏡面反射
  - **草ブロック** - Minecraft 風テクスチャ画像 (`textures/grass_block.png`) を UV マッピングで貼り付け
  - **床** - 大理石風チェッカータイル (目地・脈模様・バンプマッピング)
- **ライティング**:
  - **スポットライト** - コーン減衰付きエリアライト、ソフトシャドウ
  - **ボリュメトリックライト** - レイマーチング (32ステップ) による光の筋の可視化
  - **弱い環境光** - 暗めのムードで、スポットライトの存在感を強調
- **その他**:
  - マルチサンプル AA (4spp、サブピクセルジッター)
  - ACES トーンマッピング
  - プロシージャルテクスチャ (ノイズベース、解像度無制限) + 画像テクスチャ
  - 最大 6 バウンスのレイ追跡
  - カメラ自動周回

### sand_cube

砂ブロックのテクスチャを貼った立方体を描画。

- **レンダリング方式**: ラスタライズ (通常の頂点/フラグメントパイプライン)
- **ライティング**: Phong シェーディング (Ambient / Diffuse / Specular、砂のマット質感に合わせ Specular 弱め)
- **ジオメトリ**: 法線 + UV 座標付き 36 頂点 (6面 x 2三角形 x 3頂点)
- **テクスチャ**: `textures/sand_block_256x256.png` (256x256、Minecraft 風砂ブロック)
- **操作**: ESC で終了

## 依存ライブラリ

| ライブラリ | 用途 | 備考 |
|---|---|---|
| GLEW 2.2.0 | OpenGL 拡張ローダー | FetchContent で自動ダウンロード・静的ビルド |
| GLFW 3 | ウィンドウ / 入力管理 | MinGW 用静的ライブラリが必要 (`/usr/x86_64-w64-mingw32/lib/`) |
| stb_image | PNG/JPG デコーダー | FetchContent で自動ダウンロード (ヘッダーオンリー) |

## ビルド環境

- CMake >= 3.16.3
- C++17 / C99
- クロスコンパイラ: `x86_64-w64-mingw32-g++` (GCC 9.x)
- 実行環境: Windows (OpenGL 3.3 Core Profile)
